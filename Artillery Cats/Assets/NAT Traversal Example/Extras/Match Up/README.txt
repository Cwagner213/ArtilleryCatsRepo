The ExampleMatchUp scene contains an example implementation of Match Up combined with NAT Traversal.
Match Up is our matchmaking plugin that you can get here: http://u3d.as/10eJ

Instructions:
	1. Make sure you have imported the Match Up plugin.
	2. In Player Settings under Other Settings add Scripting Define Symbol "MATCH_UP"
		* This allows NAT Traversal to know that Match Up is installed
	3. Build and run.
	4. Press "Host" button in the build to host.
	5. Run in the editor.
	6. Press the "Search matches" button to get a list of matches.
	7. Click the button for the match you want to join.

Next Steps:

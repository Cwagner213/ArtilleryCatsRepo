The ExampleHostMigration scene contains an example implementation of Host Migration combined with NAT Traversal. 

Instructions:
	1. Build and press the "Host" button to host
	2. Run in the editor and press the "Join" button to join
	3. Press the "Disconnect" button to disconnect on the host
	4. Press the "Pick New Host" button on the client to pick a new host
	5. Press the "Start As Host" button on the client to become the new host
	6. Press the "Pick New Host" button on the old host to pick a new host
	7. Press the "Reconnect as Client" button on the old host to connect to the new host.
	8. Host has been now been migrated.

Check out the ExampleHostMigration.cs script to see how it all works.
